# Projeto ETL com Apache Hop

##  Objetivo

Demonstrar um processo de ETL (Extração, Transformação e Carga) utilizando Apache Hop com dados do Currículo Lattes e visualização no Power BI.

##  Ferramentas Utilizadas

* Apache Hop
* Docker
* DBeaver
* Power BI

##  Processo ETL

### Extração

Dados extraídos de arquivos XML do Lattes e banco de dados.

### Transformação

Tratamento dos dados com seleção, ordenação e junção (merge join).

### Carga

Dados inseridos no banco através de operações de insert/update.

##  Visualização

Os dados foram analisados no Power BI, permitindo visualizar a quantidade de artigos por ano por pesquisador.
